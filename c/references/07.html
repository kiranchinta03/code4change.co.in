   	Looping Statements
	
	looping statements  are used to execute the statements repeated times or required number or times.
	Looping means executes the statements repeated times or required number of times. In c language three types of looping statements.
	1.while loop statements
	2.do-while loop statements
	3.for loop statements
	any looping statements may contain the following
	1.initialization of  looping variable
	2.condition 
	3.increment or decrement of looping variable

	while loop statements:-  In while loop statements the body of the loop is executed repeatedly as long as he given the condition is true 
		(or)
	In while loop statements the body of the loop is executed repeatedly until the given the condition is false.
		While(condition)
		{
			The body of the loop
		}

	do-while loop statements:-  in do while loop statements the body of the loop is executed repeatedly as long as the given the condition is true.
		(or)
	In do-while loop statements the body of the loop is executed repeatedly until the given the condition is false.
		Do
		{
			The body of the loop
		}while(condition);

	For loop:-  in for loop statements the body of the loop is executed repeatedly until the given the condition is false.
		(or)
	In for loop statements the body of the loop is executed repeatedly as long as the given is true.

		For(initialization of body looping variable ; condition ; increment or decrement of looping variable)
		{
			Body of the loop;
		}

	Break:- break statement is used to transfer the control to following the inner most enclosing of while or do-while or for loop statements . 

	Nested loops:-
		If a looping statements contains another looping statements in the body of the loop is called as nested loops. They are three types
		a) Nested while loop
		b) Nested do-while loop
		c) Nested for loop

	Nested while loop:-
	While(condition)
	{
       		While(condition)
		{
		Body of the loop
       		}
	}


	Nested do-while loop:-
	Do
	{
        	 Do
         	{
		Body of the loop;
	        }while(condition);
	}while(condition);


	Nested for loop
	For(initiation of looping variable ; condition ; increment or decrement of looping variable)
	{
	     For(initiation of looping variable ; condition ; increment or decrement of looping variable)
     		{
		Body of the loop;
	      	}
	}


	SWITCH  STATEMENT 
	Switch(expression)
	{
		Case   constant 1:
		Sm1;
		Sm2;
		Break;
	
		Case   constant2:
		Sm1;
		Sm2;
		Break;

		Default:
		Sm1;
		Sm2;
	}

	Switch statement is case control statements. It allows us to make a decision from the number of choices. 
	Switch, case, default break are the keywords.

	The keyword break is used as a terminator for every case.
	During the execution time is takes the value of expression and matches with case constant. If any one case is matching 
	then the hander comes to that case and executes all statements present i.e. that case till finds the break.

	Default case is similar to the else part of multiple selection of if statement. This while be executed when none of the 
	case all matching with the expression value.
