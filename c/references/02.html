2. Character set of C -language
 

   	
	In  c-language alphabets ,digits and symbols are called as character set of c-language
	Alphabets : a to z  A to Z
	Digits	    :  0 to  9
	Symbols  :   + - * / % !@ # $ % ^ & = | \ : ; ‘ “ [ ] { } ( ) < > , . ?

	Constants :- Constant  is one which can not change its value  during the program execution
           Ex X=12  
	here  12 is constant

	variable :- Variable  is one which can  change its value during the program execution (0r) Variable is a 
	name that given to memory allocation where the constants are stored.
           Ex X=12  
  	Here x is variable
	
	Type of constants :-
	=>Integer constant
	=>Real constant
	=>Character constant
	=>String constant

	Integer constant :- A constant which is not having any decimal place.    Ex 12,10,-12,-10 etc

	Real constant :- A constant which is  having any decimal place.    Ex 12.5,10.5,-12.5,-10.5 etc

	Character constant :- if a single character is represented in single quotations is called as character constant
 	Ex    ‘k’    ‘9’    ‘$’   etc

	String constant  :- A group of characters will make  string. Strings are represented in double quotations. Ex “ravi”,  
	“anantapur”,”12-a”     “place”

	Types variables:-
	=>Integer variables
	=>Real variables
	=>Character variables
	=>String variables

	Integer variables :-  A variable  which is used to store an  integer constant  is called as integer variable
	    Ex x=10
  	X is  integer variable


	Real variables :-  A variable  which is used to store  real constants  is called as real variable
	   Ex y=10.5
	  y is  real variable


	Character  variables :-  A variable  which is used to store  character constants  is called as character  variable
	    Ex   ch=’k’
	Ch  is  character variable
	
	String variables :-  A variable  which is used to store  string constants  is called as string variable
    	Ex name=”ravi”

	Rules for naming a variable :-
	1. Variable name should not exceed 32 letters
	2. First letter of variable must be an alphabet
	3. Spaces and special symbols are not allowed with in the variable name
	4. Keywords should not be used as variable name.
