5. Operators
 

   	Operator are symbols which are mainly used in various operations. In c-language there different type of operator. 
	The data items which act upon operators are called as operands.some operators required two operands and some operators work on 
	single operands.
		1.Arithmetical operators.
		2.Relational operators.
		3.Logical operators.
		4.Increment & decrement operators.
		5.Conditional operator.
		6.Assignment operator.

	Arithmetical operators:-  These operators are used in arthimatical calculations. These operators
	Are  +,-,*,/,%.
	
	Relational operators:- These operators are used to compare two values.. These operators are used to form 
	relational conditional. there are >,<,>=,<=,= =,!=.


	Logical operator:- these operators are used to combine two or conditions. they are 
	&&	and
	||	or
	!	not

	Increment & decrement operators:-these are unary operators + + is an increment operator, - - is decrement operator.
	+ + operator is used to one increment to the value of variable. - - is used to one decrement to the value of variable.
	Suppose  i=1;
	   	 L =i+1;
	This statement can be written as  i++,++i.
	Suppose  i=1;
		 L =i-1;
	This statement can be written as  i--,--i.

	Conditional operator(?):- ?: is called as conditional operators. Simple conditional operations can be checked through conditional operator.
	Syntax:-        exp1 ? exp2 : exp3
