	In c –Language there are 4 types of statements. They are 
	1.Type declaration statement
	2.Input/Output statement
	3.Arithmetical   statement
	4.Control statement

	TYPE DECLARATION STATEMENT:-
 	This statement is used to declare type variables.  We must declare the all the variables which we used in a program. 
	This statement  is usually  written at the  beginning of a program

	Syntax :-   Data type    Variables;
	a)Declaring  integer variables
       		int   sno,m1,m2,m3,total;
	b)Declaring  real variables
       		float   avg;
	c)Declaring character variables
       		char   ncc;
	d)Declaring string variables  
      		char  name[20],course[20];

	Input statement :-  A statement  which is used to input  the data is called as input statement . 
	In C-language scanf( )   is a input statement. 
	Syn :- Scanf (“formatstring”,&variables);


	%d  is format string  for integer variables
	%f  is format string  for real variables
 	%c  is format string  for character variables
	%s  is format string  for string variables

	a)Inputting  an Integer variables :-
		Scanf(“%d%d%d%d”,&sno,&m1,&m2,&m3);
	
	b)Inputting   Real variables :-
		Scanf(“%f”,&wt);

	c)Inputting  character variables :-
		Scanf(“%c”,&ncc);

	d)Inputting  string  variables :-
		Scanf(“%s%s”,name,course);

	Out put statements :-  a statement which is used to display the output is called as  output statement.
	in c-language printf( ) is a output statement.

	SYN : - printf(“format string”,variables);
 
	Model :1

	printf(“%d”,a);
	printf(“%d”,b);
	printf(“%d”,c);
	or
	printf(“%d%d%d”,a,b,c);

	Model :2

	printf(“%d\n”,a);
	printf(“%d\n”,b);
	printf(“%d\n”,c);
	or
	printf(“%d\n%d\n%d\n”,a,b,c);
	
	
	
	Model :3

	printf(“a value=%d\n”,a);
	printf(“b value=%d\n”,b);
	printf(“c value=%d\n”,c);

	Model :4

	printf(“%d\t%d\t%d\t”,a,b,c)



	Model :5
	message printing :-
	syn  :- printf(“message”);
	ex    :- printf(“enter a,b values…………”);
	


	Assignment statement :- This statement is used for assign a constant to variable .
	Syn :-   Variable= Constant ;
  		K=10;
		Y=10.8;
		CH=’Z’;

	Arithmetical statement :- A statement which is used for doing arithmetical calculations. In Arithmetical statements 
	variables and constants are combined with arithmetical operators  (+,-,*,/,%) . These statements always returns a numeric value.
	Ex :-   c=a+b;
           	a=l*b;
           	i =(p*t*r)/100;
	Note 1:-Arithmetical calculations in between an integer and an integer will gives an integer value.
      	Ex:-   3+2=5
                3-2=1
                3*2=6
                3/2=1
	Note 2:- Arithmetical calculations in between a real and real will gives a real value.
      	Ex:-   3.0+2.0=5.0
                3.0-2.0=1.0
                3.0*2.0=6.0
                3.0/2.0=1.5
