	STRING FUNCTIONS :- 
	String functions are used to manipulate  The strings. While  using string  functions we
	 must include  the header file    #INCLUDE<STRING.H>

	STRING FUNCTIONS ARE  
	A)STRLEN (  )
	B)STRLWR( )
	C)STRUPR(  )
	D)STRCPY( )
	E)STRCAT( )
	F)STRCMP( )
	G)STRREV ( )

	1. STRLEN( ) :- This function is used to find the Length of a string
	SYN  :-  strlen(string)
	Ex:-	strlen( name );

	2. Strlwr( ) :- This function is used to convert the string into lower case.
	Syn  :-  strlwr(string)
	Ex:-	strlwr( name );

	3. Strupr( ) :- This function is used to convert the string into upper case.
	Syn  :-  strupr(string);
	Ex:-	strupr( name );

	4. Strrev( ) :- This function is used to convert the string into  reverse.
	Syn  :-  strrev(string);
	Ex:-	strrev( name );

	5. Strcpy( ) :- this function is used to copy a string into another string
	Syn  :-  strcpy(target string,source string)
	Ex:-	strcpy( name2,name1);

	6. Strcat( ) :- this function is used to combine two strings
	Syn  :-  strcat(target string,source string)
	Ex:-	strcat( name1,name2);

	7. Strcmp( ) :- this function is used to compare  two strings.if two strings are same then this 
	Function return zero. 
	Syn  :-  variable=strcmp(target string,source string)
	Ex	k=strcmp( name1,name2);
