   	Arrays
	Array is a variable which is used to store multiple data items of same  Type.

	Arrays are two types.
	1.Single dimensional arrays
	2.Multi dimensional arays

	SINGLE DIMENSIONAL ARRAYS:-
	In array declaration if  we specify only one dimension is called as single dimensional arrays

	MULTI DIMENSIONAL ARRAYS:-
	In array declaration if  we specify  more than one dimensions  is called as multi  dimensional arrays

	SYNTAX OF  SINGLE DIMENSIONAL ARRAYS

	DATATYPE        ARRAYNAME  [SIZE];

	In the above syntax data type indicates the type data items that stored in array variable.
	The size indicates no of data items that stored in array variable.
	EX    int  x[10];
        	float  y[10];
        	char  name[20];

	Example 1  tells that  in array variable x  we can store 10 Integer values. 
	Those values are stored in x[0],x[1],x[2]………..x[9] respectively.

	Example 2  tells that  in array variable  y  we can store 10 real  values. 
	Those values are stored in y[0],y[1],y[2]………..y[9] respectively.

	Example 3  tells that  in array variable  name  we can store  20 characters . 
	Those characters  are stored in name[0], name[1], name[2]………..name[19] respectively.

	INTIALIZING ARRAY :-
	EX:-   int x[5]={10,20,30,40,50};
          	float   y[5]={1.1,2.1,3.1,4.1,5.1};
          	char   name[5]={‘r’,’a’,’v’,’i’,’\0’};
