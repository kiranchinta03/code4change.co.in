   	DataFiles
	Data file is used to store the data permanently is secondary storage devices.
	The data that stored ij secondary storage devices can retrieved whenever necessary.

	File operations:-
	1.creating a data file 
	2.storing of data into a data file
	3.closing a data file
	4.opening of a data file
	5.reading of data from data file
	6.closing a data file

	Before file operations we must a declare a file pointer. A file pointer can be declared by using the following syntax.

	Syntax:-	File     *pointer ;
	Ex:-		File      *p ;

	Creating a data file:-
	Fopen( ):- This function is used to create or to open a data file.
	Syntax:-    file pointer = Fopen( “ filename “, “ mode “);
	
	Ex:-   P = fopen( “ stc “, “ w ”);
	Ex:-   P = fopen( “ stc “, “ r ”);
	Ex:-   P = fopen( “ stc “, “ a ”);

	Storing of data into a data file:-
	Putw ( ) :- This function is used to store the integer data into a data file.
	Syntax:-   putw(integer variable,filepointer);
	Ex:-	   putw(n,p);


	Putc ( ) :- This function is used to store the character data into a data file.
	Syntax:-    putc(integer variable,filepointer); 
	Ex:-	   putc (ch,p);

	Fprintf ( ) :- This function is used to store mixed data into a data file.
	Syntax:-   fprintf(file pointer, “ formatting string “ , variable );
	Ex:-	   fprintf(p , “ %d%f%s “ bno,bp,bname);

	Reading of data from a data file:-

	Getw ( ):-  This function is used to read the integer data from a data file.
	Syntax:-   integer variable = getw ( filepointer);
	Ex:-	   n=getw(p);

	Getc ( ):- This function is used to read the character data from a data file.
	Syntax:-   character variable = getc ( filepointer);
	Ex:-	   ch=getc(p);

	Fscanf ( ):- This function is used to read mixed data from a data file.
	Syntax:-   fscanf(filepointer, ” formatting string “ , &variable);
	Ex:-	   fscanf(p, ” %d%f%s “ , &bno,&bp,bname);

	Closing a data file:-
	Fclose ( ):-  This function is used to close a data file.
	Syntax:-   fclose ( filepointer);
	Ex:-	  fclose(p);
