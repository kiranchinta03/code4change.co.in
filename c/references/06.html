   	Control statements
	C Language must be able to perform different sets of actions depending on the circumstances. A C program is a set of 
	statements which are normally executed sequentially in the order in which they appear. This happens when no options  
	or no repetitions of certain calculations are necessary. However, in practice, we have a number of situations where 
	we may have to change the order of execution of statements based  on certain conditions ,repeat a group of statements 
	until  certain specified conditions are met. This involves a kind of  decision making to see whether a particular condition 
	has  occurred or not and then direct the computer to execute certain statements accordingly. 
	
	In c language there are 2 types of control statements:
		Branching statements
		Looping statements
		Case control statements

	BRANCHING STATEMENTS ARE TWO TYPES 
	A)Conditional control statement
	B)Un conditional control statement

	CONDITIONAL  STATTEMENTS ARE 4 TYPES 
	1.if statement
	2.if –else statement
	3.nested if statement
	4.Else if ladder


	UN CONDITIONAL  STATTEMENT
	1.Goto statement


	CASE CONDITIONAL  STATEMENT 
	1. switch statement


	The IF Statement:
	  	The IF Statement is a powerful decision making statement and is used to control the flow of execution of statements. 
		The keyword 'IF' tells the compiler that what follows is a decision control instruction. The condition following the  
		keyword 'If' is always enclosed within a pair of Parenthesis.  It allows the computer to evaluate the expression  
		first and then depending on whether the value of the  expression (relation or condition) is 'TRUE' (non-zero) or 
		FALSE (zero),it transfers the control to a particular statement. This point of program has two paths to follow one 
		for the TRUE condition and other is for FALSE condition.


	The General form of IF statement is look like this: 
            	 if(condition)
		statement1;
     
 	(or)
		if(condition)
		{
		statement1;
		statement2;
		statement3;
		---
		--
		}

	IF- ELSE Statement:
		If-.Else statement is an extension of the simple IF statement. In if-else statement the condition is tested if the 
		given condition is true then it executes true block statements other wise it executes false block statements.
 	The general form is:

               if(test condition) 
               { 
                 True-Block statement(s) 
               } 
               else 
               { 
                 False-Block statement(s)   
               } 

		In if-else statements the condition is tested if the give condition is true then it executes true block statements 
		other wise it executes false block statements.

	Nested if statements:-
		If an if statements contains another statements either if block or in else block is called as nested if statements
	


	Syntax:-

		If(condition)
		{
			Sm1;
			Sm2;
			Sm3;
		}
		Else
		{
		If(condition)
			Sm1;
			else
			Sm2;
		}

	(or)

		If(condition)
		{
			If(condition)
			Sm1;
			Else;
			Sm2;
		Else
		{
			Sm1;
			Sm2;
		}


	ELSE IF ladder:-
		If(condition)
		Sm1;
		Else if(condition)
		Sm2;
		Else if(condition)
		Sm3;
		Else if(condition)
		Sm4;
	
	GOTO Statements:-
		Goto is a unconditional control statements. In goto statements the control is transferred unconditionally to the 
		location of local label specified by an identifier.

	Syntax:-   
	Forward jumping 						back ward jumping
		Goto label							label:
		Sm1;								sm1;
		Sm2;								sm2;
		Sm3;								sm3;
		Label:								goto label;

	Relational and logical conditions:-
	A conditional which is formed by using relational operator is called as relational conditional.
	Relational operator are >,<,>=,<=,==,!=. relational operators are used to compare two values.
	Ex
     		 (a>b), (a=100), (a<=100)

	A condition which is formed by using logical operators is called as logical condition.
	Logical operators are && (and) , ||(or), !(not). Logical operators are used to combine two or more relational condition.
	Ex 	((a>b)&&(a>c))
		((a.b)||(a>c))
		!(a>b)
