   	Structures
	structure is a user defined data type.in structure variable we can store different type of data items. For example 
	if u want store a book details or student details or emp details  then we can make use of structures

	The general form of structure mainly contain two parts.
	1.Structure defination
	2.Structure variable declaration

	Structure defination:-
	Struct     structure name
	{
	Structure  element1 ;
	Structure  element2 ;
	--
	--
	} ;

	Ex :-	struct book
      		{
         	int bno;
        	float bprice
         	char  bname[20];
          	};

	Structure variable declaration :
	Syn :-  struct   structurename structure variables;
	Ex:-     struct    book  b1,b2,b3;


	Accessing  of  structure   elements :-
	Structure variable . Structure element
	Ex	b1.bno
		B1.bp
		B1.bname

	Intilizing structure variables :-
	Struct structure name  variable ={constant1,constant2,………};
	Struct    book      b1=  {1,100.50,”msoffice”};

	Special features of structures :-
	We can assign  a structure variable to another structure variable.

	Struct    book  b1={1,100.50,”msoffice”};
	Struct    book    b2;
	B2=b1;
