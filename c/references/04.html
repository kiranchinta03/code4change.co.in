   	Structure of C-prg :-
	Preprocessor statements
	Main( )
	{
	Type declaration statements;
	Input statements;
	Arithmetical statements;
	Output statements;
	}
	Preprocessor statements :- A statement which starts with symbol # is called as preprocessor statement.
	Ex :- #include<stdio.h>
         #include<conio.h>

	Here #  is a preprocessor. These statements are execute before processing. 

	#include<stdio.h> :- this statement includes the header file stdio.h into a program.
	#include<conio.h> :- this statement includes the header file conio.h into a program.

	Header files :- A header file  contains function definitions. Stdio.h contains  scanf (), printf( )   definitions. Conio.h contains clrscr( ),getch( ) definitions.
	in  structure of program  main () is a function. It indicate that program execution. Begins from main()

	Symbolic constants:-
	Using a meaning name either to a constant value or to any expression is called as symbolic
	Constant .it can be defined using a preprocessor statement.
	Syntax: #define macro name    constant value
             #define pie                    3.14
             #define p                        print
