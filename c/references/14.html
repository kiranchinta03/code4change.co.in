   	Unions
	union is a user defind data type. It is just like structures in syntax. In structure variable we can store only one of union items. 
	The total memory allocated to structure variable is sum of memory allocated all structure elements. But total memory allocated to 
	union variable is maximum memory of union element.

	The general form of structur mainly contains two parts.
	1.union definition.
	2.union variable declaration.

	Union definition:-
	Union      union name
	{
        	Union      element1;
	        Union      element2;
        	Union      element3;
	-	- - - - -
	-	 - - - - -
	-	- - - - - 
	};

	Ex:-    union     book
		{
	        	Int      bno;
		        Float   bprice;
		        Char    bname [20];
		};

	union variable declaration
	syntax:- union      union name       union variable;
	ex:-        union      book		 b1,b2,b3;

	accessing of union elements:-
	syntax:-   union variable  . union element
	ex:-	   b1. bno
		   b2.bp
		   b3.bname
