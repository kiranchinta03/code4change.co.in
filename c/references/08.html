   	Functions
	Function is a sub program of a main program. If the main program is divided in to sub programs those sub programs are called as functions (or) procedures (or) modules.

	“The method of dividing a program into sub programs is called as structure oriented programming   C-language follows structure oriented programming . So C-language is called as structure oriented programming language”.

	Functions are  two types :
	1.LIBRARY FUNCTIONS
	2.USER DEFINED FUNCTIONS

	LIBRARY FUNCTIONS :-  A function definition which is pre defined to c-language compiler  is called as library functions.

	EX :- SQRT( ), POW ( ), SCANF( ), PRINTF( ) ,GETS( ),PUTS ( ) ,TOUPPER( ),TO LOWER()

	Function 					purpose

	Sqrt()		This function is used to calculate square root values
			Area=sqrt(s*(s-a)*(s-b)*(s-c))

	Pow( )		This function is used to calculate power values
			C=ab
			C=pow(a,b);

	Toupper( ) 	This function is used to convert a letter in to upper case
			Letter=toupper(letter);

	Tolower( ) 	This function is used to convert a letter in to lower case
			Letter=tolower(letter);

	Getchar( )	this function is used toinput a letter through key board
			Letter=getchar( );

	Putchar( )	this function is used to print a letter through key boad
			Putchar(letter);

	Gets ( )	this function is used to input a line of text through key board.
			Gets(name);

	Puts( )		this function is used to print a string through key board.
			Puts(name);



	USER DEFINED FUNCTIONS:-  A function definition which is  defined By the user   is called as user defined  functions.
	EX : SUM( ) , AREA( ), INTEREST( ), FACTORIAL( )

	USES OF FUNCTIONS:-
	1.By using functions we can reduce the program length
	2.By using functions we can increase the program clarity
	3.Error debugging is easy

	THE GENERAL FORM OF A FUNCTION MAINLY CONTAIN 3 PARTS
	1.Function Declaration
	2.Function Calling
	3.Function Definition

	FUNCTION DECLARATION :- 
	SYNTAX :-   RTYPE    FNAME (TYPE OF ARGUMENTS) ;

	RTYPE:- Means function return value type. Here  void or int or float May be used as Rtype.
	Void is a key word . It means function will return nothing

	FNAME:- fname means function name. Any valid name be given as Function name

	TYPE OF ARGUMENTS:- type of  arguments  means type of variables
	EX :-       void  sum(  );
        	    void  sum (int, int);
		    int   sum( int, int);


	FUNCTION CALLING:-  When ever we call a function in main program the control will come to function definition and 
	executes the function definition and the control will return to main program.
	SYNTAX :-   FNAME (ACTUAL ARGUMENTS);
 
	ex  	Sum ( );
     		Sum(a,b);
		C=Sum(a,b);
	
	NOTE :- The arguments which exist in function  calling are called as 
	Actual  arguments.

	CALL BY VALUE:- In function calling if we pass the values of actual arguments  to function definition  is  called as call 
	by value method The second  and third methods (function with args ,no return value, function with args with return value) 
	follows call by value method
		EX :-  	 SUM(A, B);
		         C=SUM(A,B);

	CALL BY REFERENCE:-  In function calling if we pass the addresses  of actual arguments  to function definition instead of  
	passing their values  is  called as call by reference  method
		EX:    SUM(&A,&B);
		       C=SUM(&A, &B);
