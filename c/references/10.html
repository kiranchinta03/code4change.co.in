   	Storage classes
	If you want to a declare a variable we have to specify not only its data type but also its storage class.
	In c-language there are 4 type of storage classes.
	1.Automatic storage class
	2.static storage class
	3.register storage class
	4.external storage class

	In c-language a variable can be declared as follows
	Storage class data type variable;

	Ex:- 
		Auto     int  x;
		Static    int  y;
		Register  int  z;
		Extern   int  k;

	Note :- in variable declaration if we do not specify storage class then c-compilar takes default automatic storage class as storage class.

	Storage class of variable indicates the following:-
	1.Storage
	2.Default intial value
	3.Scope of the variable
	4.Life of the variable
	 

	If  u want declare a variable as automatic storage class then use key word “auto” as storage class

	If  u want declare a variable as static storage class then use key word “static” as storage class

	If  u want declare a variable as register storage class then use key word “register” as storage class

	If  u want declare a variable as external storage class then use key word “extern” as storage class
