2. JAVASCRIPT Statements
 

   	
	JS Statements
		When writing your own scripts it's very useful to know some of the most common JavaScript statements. With if() 
		you can check conditions, while for() allows you to go through a set of data and do the same with each piece of data. 
		Both are very common statements and you should know how they work. 
		
		JavaScript Comments: We are having two types of comments in javascript.               
			1.Single line comments                
			2.Multi line comments 	
		
		single line comment
			 To create a single line comment in JavaScript, you place two slashes "//" in front of the code or text you wish to have 
			 the JavaScript interpreter ignore. When you place these two slashes, all text to the right of them will be ignored, until 
			 the next line.   
			
			<script type="text/javascript">
			<!-- 
			// This is a single line JavaScript comment 
			document.write ("I have comments in my JavaScript code!"); 
			//document.write ("You can't see this!"); //--> 
			</script> 
			

			
		Multiple line comments
			Although a single line comment is quite useful, it can sometimes be burdensome to use when disabling long segments of code 
			or inserting long-winded comments. For this large comments you can use JavaScript's multi-line comment that begins with 
			/* and ends with */. 
			
			<script type="text/javascript">
			<!-- 
			document.write ("I have multi-line comments!"); 
			/*document.write ("You can't see this!"); 
			document.write ("You can't see this!"); 
			document.write ("You can't see this!"); 
			document.write ("You can't see this!"); 
			document.write ("You can't see this!"); 
			document.write ("You can't see this!"); 
			document.write ("You can't see this!");*/ 
			//-->
			</script>
