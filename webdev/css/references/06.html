	Properties and their Values
	
	Color Property
	Hexadecimal Notation : p{ color:#FF0000; } ,  p{ color:#F00; } 
	Functional Notation : p{ color:rgb(255,0,0); } , p{ color:rgb(100%,0%,0%); } 
	
	BACKGROUND PROPERTIES 
	Property   					value 
	------------------------------------------------------------- 
	background   				<color><image><repeat><attachment7gt;<position> 
	background-attachment 		fixed 
	background-color  			color name / hexa decimal notation 
	background-image  		path file 
	background-position  		Sets the offsets left and top 
	background-repeat  		repeat, repeat-x, repeat-y, no-repeat. 
	
	FONT  
	Property   			value 
	--------------------------------------------------- 
	font-family   		arial, times new roman etc. 
	color        		color name / hexa decimal notation 
	font-style 		italic. 
	font-weight 		bold
	font-size   		px, %. 
	font-variant   		smallcaps. 
	
	TEXT   
	Property   						value

	LINKS  
	Property   				Description 
	--------------------------------------------------- 
	a:link{..}   			set link properties. 
	a:visited{..}   			User visited an element 
	a:active{...}  			User activates an element (e.g. mouse down). 
	a:hover{...}   			User designates an element (e.g. mouse over).

	text-align   				left, right,center
