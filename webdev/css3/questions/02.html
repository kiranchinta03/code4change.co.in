	14. What do you mean by responsive design?
	It’s about making websites work on different platforms/devices. Different devices with different sizes and different capabilities. 
	Responsive design is about taking one code base and making it work for all of them. Media queries used for achieving this.

	17. What is the CSS3 background size Property? How to use it?
	The background­size property specifies the size of the background image.
	
		.myClass {
		background:url(background.jpg);
		-moz-background-size:80px 60px; /* Firefox 3.6 */
		background-size:80px 60px; /* or we can do  background-size:100% 100%;*/
		background-repeat:no-repeat;
		}

	19. What CSS3 box­sizing Property?
	The box­sizing property is used to tell the browser what the sizing properties (width and height) should include. box­sizing to 
	“border­box” forces the browser to render the box with the specified width and height, and place the border and padding inside the box.
	
	20. What is CSS3 backface­visibility Property?
	The backface­visibility property defines whether or not an element should be visible when not facing the screen.This property is useful
	 when an element is rotated, and you do not want to see its backside.
	
	21. What are the values for postion property in css? What is the difference?	
	Static: Default value. Elements render in order, as they appear in the document flow 
	Absolute: The element is positioned relative to its first positioned (not static) ancestor 	element
	Fixed: The element is positioned relative to the browser window
	Relative: The element is positioned relative to its normal position, so “left:20” adds 20 pixels to the element’s LEFT position.
